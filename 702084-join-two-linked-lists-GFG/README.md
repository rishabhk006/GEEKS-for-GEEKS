# [Join Two Linked Lists](https://www.geeksforgeeks.org/problems/join-two-linked-lists/1?page=6&category=Linked%20List&sortBy=submissions)
## Easy
Given heads of two singly linked lists, head1 and head2.&nbsp; Append the second list to the end of the first list. After performing this operation, return head of the new list .
Examples:
Input: head1: 1 -&gt; 2 -&gt; 3, head2: 4 -&gt; 5 &nbsp;Output: 1 -&gt; 2 -&gt; 3 -&gt; 4 -&gt; 5Exaplanation: After joining both lists we get 1 -&gt; 2 -&gt; 3 -&gt; 4 -&gt; 5
Constraints:1 ≤ number of nodes&nbsp; ≤ 1051 ≤&nbsp; value of nodes ≤ 105
