# [Palindrome String](https://www.geeksforgeeks.org/problems/palindrome-string0817/1?page=3&sortBy=submissions)
## Easy
Given a string s, return true if the string is a palindrome. Otherwise, return false. 
A string is considered a palindrome if it reads the same forwards and backwards.
Examples :
Input: s = "abba"
Output: true
Explanation: "abba" reads the same forwards and backwards, so it is a palindrome.
Input: s = "abc" 
Output: false
Explanation: "abc" does not read the same forwards and backwards, so it is not a palindrome.
Constraints:1 ≤ s.size() ≤ 106The string&nbsp;s contains only lowercase english letters (a-z).
